
## System Configuration Files with i3 and Arch for minimal desktop experience

### Configure

    git clone https://github.com/raz0229/.dotfiles.git
    cd .dotfiles/
    chmod +x configure.sh
    ./configure.sh
#### External Dependencies:
``clipit``
``pavucontrol``
``rofi``
``i3status-rust``
``gsimplecal``
``feh``
